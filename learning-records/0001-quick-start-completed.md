## Summary

学习了 mihomo-manager 的基本操作流程：编译安装 → 查看状态 → 安装 mihomo → 启动/停止 → 升级 → 卸载。同时掌握了 TUI 的键盘快捷键布局。

## Key insights

- 所有操作都通过 `mihomo-manager <command>` 统一入口
- 安装/升级/卸载需要 sudo，日常启停不需要
- TUI 是默认入口，CLI 适合脚本化
- 退出码 0/1/2 区分三种状态
