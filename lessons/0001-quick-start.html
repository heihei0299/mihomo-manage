Lesson 1: mihomo-manager 快速入门

完成本节后你将能用一条命令完成 mihomo 的安装、配置和日常启停。

准备工作

编译当前项目：

go build -o mihomo-manager .
sudo cp mihomo-manager /usr/local/bin/
注意：部分操作（安装、卸载、系统服务注册）需要 sudo 权限。

1. 查看状态

任何时候都可以通过 status 命令查看当前 mihomo 的安装和运行状态：

mihomo-manager status

输出示例：

mihomo: not installed    ← 尚未安装
mihomo: running v1.19.27 ← 正常运行中
mihomo: stopped v1.19.27 ← 已停止

退出码含义：0 = 运行中，1 = 已停止/失败，2 = 未安装。

2. 安装 mihomo

安装最新版：

sudo mihomo-manager install

安装指定版本：

sudo mihomo-manager install v1.19.27

安装过程分 5 个阶段，每个阶段都有进度输出：

  [fetch] Downloading mihomo v1.19.27
  [fetch] Download complete
  [deploy] Deploying binary
  [bootstrap] Creating config directory
  [register] Registering system service
  [start] Starting mihomo
mihomo installed successfully

3. 查看版本列表

mihomo-manager versions

从 GitHub Releases 获取最近 5 个可用版本：

v1.19.27
v1.19.26
v1.19.25
v1.19.24
v1.19.23

4. 升级 mihomo

升级到最新版：

sudo mihomo-manager upgrade

升级到指定版本：

sudo mihomo-manager upgrade v1.19.26

升级流程：下载新版本 → 停止当前实例 → 备份旧二进制 → 替换 → 启动。如果启动失败会自动回滚。

5. 启停控制

启动：mihomo-manager start — 启动代理
停止：mihomo-manager stop — 停止代理
重启：mihomo-manager restart — 完全重启
重载配置：mihomo-manager reload — 不重启进程，仅重载配置

6. TUI 交互界面

不加任何参数运行即启动 TUI：

mihomo-manager

TUI 中的操作键：

  • 1 Start / 2 Stop / 3 Restart / 4 Reload
  • 5 Upgrade / i Install
  • r 刷新状态 / q 退出

7. 卸载

sudo mihomo-manager uninstall              # 删除所有
sudo mihomo-manager uninstall --keep-backup # 保留配置备份

后续

配置管理（订阅设置、分流规则编辑）等功能可以通过 mihomo-manager subscription set、mihomo-manager config preview 等 CLI 命令操作，详见 mihomo-manager --help。
